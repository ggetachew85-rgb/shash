# Shash v2

Personal life command center — daily plan, money, health, career, and goals in one place.
React + Vite. Data is stored on-device (localStorage). Back up/restore via the gear menu.

## Run locally
npm install
npm run dev

## Deploy
Push to GitHub, import the repo in Vercel. Vercel auto-detects Vite, builds, and serves it.
On iPhone: open the Vercel URL in Safari → Share → Add to Home Screen.
